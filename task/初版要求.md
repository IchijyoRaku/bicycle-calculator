# 需要改进的点

## UI界面

整个UI界面就按普通APP经典样式，下面放两个导航按钮，分别是计算结果（主页面）和数据设定（设置页面），图标你自行确定

功能顶部标题用下拉菜单进行切换，默认为`计算器功能`

## 数据输入

1. `人车总质量`改名为`人车质量`，类型从下拉候选改为数字输入
2. `常用踏频`，类型从下拉候选改为数字输入，单位显示为r/min
3. `牙盘规格`、`飞轮规格`、`轮组规格`、`常用踏频`、`人车质量`这几个参数归为`常用参数`组，默认值分别为50-34T，12S 11-34T，700x28C，90，78。另外`牙盘规格`、`飞轮规格`、`轮组规格`需要支持自定义值的添加
4. 飞轮规格增加一组预设，数据为11-12-15-17-19-21-23-25-28
5. `滚阻系数`、`传动效率`、`自定义CdA`归为`其他参数`组，另外再加上其他几个参数，`空气密度`：默认值1.2、`修正系数`：默认值，用于最终风阻功率的乘算倍率
6. 然后整个数据输入板块都放到`数据设定`页面里，因为这部分不用经常改动

## 计算结果

### 计算器功能（新增板块）

1. 根据功率、坡度计算速度
2. 根据速度、坡度计算功率
3. 根据选定牙盘齿数、飞轮齿数、踏频（不填入就使用默认值）计算理论速度
4. 根据目标速度、牙盘齿数、踏频计算所需飞轮齿数

### 功率-车速曲线

1. 坡度选择范围从-10%-10%扩大到-15%-25%
2. 默认只显示手变位曲线，其它曲线需要手动勾选才显示

### 踏频-车速曲线

1. 增加一个纯数据显示模式，只显示默认踏频下，不同飞轮齿数下的速度
