## 当前限制
- 当前仓库原本没有 Android 工程，因此这版是从零搭建的初始可运行骨架
- 图表为 Compose Canvas 原生简化版，还不是最终精修版
- 还没有补应用图标、签名配置、release 打包脚本细节
- 当前环境下 `gradle-wrapper.jar` 未成功自动下载，因此命令行直接执行 `gradlew` 可能还需要你本地补齐 wrapper 或直接通过 Android Studio 同步
- 当前机器只检测到 Java 8，实际构建 Android 工程需要 `JDK 17`
