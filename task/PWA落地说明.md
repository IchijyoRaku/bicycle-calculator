# PWA 落地说明

## 结论
当前项目已经切到 `PWA` 路线，不再使用 `Gradle`、`Android Studio` 或原生 Android 构建链。

## 你现在能得到什么
- 手机上可以通过浏览器“添加到主屏幕”安装成 App
- 首次访问后可离线打开核心页面
- 不需要安装 Android SDK
- 不需要同步 Gradle

## 你怎么预览
1. 在电脑上用浏览器打开当前 `index.html`
2. 用手机同一局域网访问这个站点
3. 在 Chrome 里选择“添加到主屏幕”
4. 之后像普通 App 一样从桌面打开

## 注意
- 如果直接双击 `index.html`，service worker 可能无法正常工作，建议使用本地静态服务器
- 当前仍依赖现有静态资源文件
- 后续如果要做真正原生 APK，再单独重新评估技术路线
